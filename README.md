### Kurulum Adımları

1. Projeyi bilgisayarınıza indirin:

```bash

git  clone [git@github.com:ramiko2/HappyPulse.git]

```



2. Proje klasörüne gidin:

```bash

cd [HappyPulse]

```



3. Gerekli paketleri yükleyin:

```bash

flutter  pub  get

```



4. Uygulamayı başlatın:

```bash

flutter  run

```
